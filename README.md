# Library
The simulating software for library staff. (Only in console)

Options:
 - 0 - Exit the program
 - 1 - Adding a book
 - 2 - Addition of a magazine / newspaper
 - 3 - Display of available books
 - 4 - Display of available magazines / newspapers
 - 5 - Delete the book
 - 6 - Delete the magazine
 - 7 - Add reader
 - 8 - Display readers
 - 9 - Search for a book
